# /plan-with-cc - プラン作成

## 概要
新しい機能やタスクの実装計画を作成します。Claude Code (Worker) に依頼する前の準備。

## 実行内容

1. **要件ヒアリング**
   - 何を作りたいか
   - 誰が使うか
   - 期待する動作

2. **既存コードとの関連調査**
   - 影響するファイル
   - 既存のパターンとの整合性
   - 再利用可能なコンポーネント

3. **Plans.md への追加**
   - 新しいタスクセクションを作成
   - サブタスクに分解
   - 優先度の設定

4. **CI/CD 考慮**
   - 必要なテストの特定
   - ビルドへの影響確認

## 出力形式

```
📋 新規タスク: [タスク名]

📝 要件:
- [要件1]
- [要件2]

📁 影響ファイル:
- frontend/src/mastra/agents/[agent].ts
- frontend/src/app/api/[endpoint]/route.ts

✅ サブタスク:
1. [ ] [タスク1]
2. [ ] [タスク2]
3. [ ] [タスク3]

🔄 CI/CD チェック:
- [ ] ESLint 通過
- [ ] TypeScript 型チェック通過
- [ ] ビルド成功
```

## 使用例

Cursor で「OCRAgent を実装したい」と言うと実行
