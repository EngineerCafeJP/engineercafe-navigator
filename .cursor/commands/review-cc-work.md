# /review-cc-work - Claude Code の作業レビュー

## 概要
Claude Code (Worker) が完了した作業をレビューします。

## 実行内容

1. **変更内容の確認**
   - 変更されたファイル一覧
   - 差分の概要
   - コミットメッセージ

2. **CI/CD 結果の確認**
   - すべてのチェックがグリーンか
   - 失敗があれば詳細を表示

3. **コードレビュー**
   - コーディング規約の遵守
   - セキュリティ上の問題
   - パフォーマンスへの影響

4. **Plans.md の更新**
   - `cc:DONE` マーカーに変更
   - `pm:確認済` を追加
   - 次のタスクを特定

## 出力形式

```
🔍 レビュー結果: [タスク名]

📁 変更ファイル (X件):
- frontend/src/mastra/agents/router-agent.ts (+50, -10)
- backend/agents/router.py (新規)

✅ CI/CD ステータス: グリーン
- ESLint: ✅
- TypeScript: ✅
- Build: ✅

📝 レビューコメント:
- [コメント1]
- [コメント2]

🎯 判定: 承認 / 修正依頼

💡 次のアクション:
- [次に依頼するタスク]
```

## 使用例

Cursor で「Claude Code の作業をレビューして」と言うと実行
