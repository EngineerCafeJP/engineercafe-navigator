# /start-session - セッション開始

## 概要
新しい開発セッションを開始します。プロジェクトの現状を確認し、作業の準備を整えます。

## 実行内容

1. **プロジェクト状態の確認**
   - Plans.md の現在のステータス
   - 未完了タスクの一覧
   - オープン PR の確認

2. **CI/CD ステータス確認**
   - 最新の CI 実行結果
   - 失敗があれば概要を表示

3. **ブランチ状態の確認**
   - 現在のブランチ
   - main との差分

## 出力形式

```
📍 現在のプロジェクト状態
- 現在のフェーズ: [フェーズ名]
- 未完了タスク: [数]件
- オープン PR: [数]件
- CI ステータス: ✅/❌

💡 推奨アクション:
1. [最優先タスク]
2. [次のタスク]
```

## 使用例

Cursor で「開発を始める」「今日の作業を確認」と言うと実行
